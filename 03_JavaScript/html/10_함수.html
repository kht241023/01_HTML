함수(function)

        - 목표하는 결과를 만들기 위한 코드의 묶음

        - 코드를 {} (중괄호, 블록)으로 묶어 작성하고
            붙여진 이름을 호출해서 사용
        
        * 함수 선언
        * 함수 정의
        * 함수 호출

        [기본 함수 형태]
        function 함수명() { //함수의 이름 선언
            //함수 정의
        }

        함수명(); //함수를 호출

        함수란 ? 특정적인 기능을 동작하고자 할 때 
                특정 기능을 묶어서 만드는 명칭
        
        예 )  
        
        function 전자레인지 작동시작("몇 초 인가요?"){
            1. 전자레인지 코드를 꼽는다.
            2. 전자레인지 ("몇") 초를 선택한다.
            3. 전자레인지를 ("몇")초만큼 실행한다.
            4. 전자레인지 ("몇")초가 다 되면 종료한다.
        }

매개 변수(Parameter), 전달 인자(Argument)

        * 媒介變數(매개변수, parameter)
        - 함수 선언 시 () 안에 작성되는 변수
            -> 전달 인자로 부터 받은 값을 저장
            -> 함수 정의 부분에서 사용 가능

        媒 중매   매  중개하거나 연결한다
        介 개입할 개  무엇인가 사이에 끼어있는 상태
        變 변할   변  변화하고 바뀐다는 의미
        數 셀     수  수 숫자 변수 라는 의미

        중간 역할을 하거나 전달하는 변수

        * 傳達 引子(전달 인자, Arguments)
