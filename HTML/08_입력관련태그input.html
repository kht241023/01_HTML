입력관련태그 input 편

        input type 종류
        글자    = text
        비밀번호 = password
        이메일  = email
        전화번호 = tel
        숫자 및 범위 = number
        범위(슬라이더) = range
        파일 = file
        시간 / 주 / 월 = time / week / month
        체크박스 = checkbox
        라디오 = radio
        제출 / 리셋 / 숨김 = submit / reset / hidden
